# Elephant-Tusk-Analysis
An R-based statistical analysis of morphological shifts and evolutionary pressure in African elephants.
# African Elephant Morphology Analysis

## Project Overview
This project investigates the morphological shifts in African elephant (*Loxodonta africana*) tusk lengths over a 40-year period (1966-68 vs. 2005-13) to evaluate the evolutionary impacts of selective pressure from ivory poaching. 

## Skills & Tools Used
* **Language:** R
* **Libraries:** `tidyverse`, `ggplot2`, `car`
* **Techniques:** Data Wrangling, Exploratory Data Analysis (EDA), Assumption Diagnostics (Shapiro-Wilk, Levene's Test)
* **Statistical Modeling:** Independent Samples t-test, Simple Linear Regression
* **Reporting:** R Markdown for reproducible research

## Results
The analysis revealed a statistically significant reduction in average tusk length over time, demonstrating rapid, human-induced directional selection. While age remains a strong predictor of tusk growth, the baseline growth potential has shifted dramatically downward in modern populations.
